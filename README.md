# Job Analysis Dashboard using Power BI

## Project Overview
This project is an interactive Power BI dashboard created to analyze AI & ML job trends, hiring insights, company requirements, and location-based opportunities. The dashboard provides meaningful visual insights using data visualization and business intelligence techniques.

This project was created as part of my learning journey in Power BI and Data Analytics to improve my dashboard design, data cleaning, visualization, and analytical skills.

---

## Dashboard Pages

### 1. AI & ML Job Report Overview
Provides an overall analysis of job postings, hiring trends, required skills, and important KPIs related to AI & ML jobs.

### 2. Location Insights
Analyzes job opportunities based on different locations and identifies areas with higher hiring demand.

### 3. Company & Role Insights
Shows company-wise hiring patterns, popular job roles, and role-based opportunities.

### 4. Student Hiring Insights
Focuses on student-friendly opportunities, entry-level hiring trends, and skill requirements for freshers.

---

## Features

- Interactive Power BI dashboard
- Dynamic slicers and filters for real-time interaction

- Company-wise and location-wise analysis
